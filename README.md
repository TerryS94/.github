# Default community files for @TerryS94

This repo contains default issue/PR templates, security policy, and other community files that apply to my public repos unless a repo overrides them.

Files:
- `FUNDING.yml` - sponsor links
- `SECURITY.md` - responsible disclosure
- `ISSUE_TEMPLATE/` - bug/feature/question templates
- `PULL_REQUEST_TEMPLATE.md`, `CODEOWNERS`, `dependabot.yml`
